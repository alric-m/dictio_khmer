== DESCRIPTION

This Rails app aims to provide a platform validating your skills obtained in Khmer (from English or French).

* Ruby version => 2.2.3

* Rails version => 4.2.5.2

== HOW TO

* <tt>bundle</tt>

* <tt>rake db:seed</tt>

* <tt>rails s</tt>
